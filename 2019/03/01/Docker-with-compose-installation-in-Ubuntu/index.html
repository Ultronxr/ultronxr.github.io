五、docker常用命令

作用 命令 参数含义
登录docker docker login 无
查看当前正在运行的所有镜像 docker ps 无
查看当前正在运行的所有镜像的id docker ps -q 无
查看所有镜像 docker ps -a 无
查找镜像 docker search <image> image：镜像名称
下载镜像 docker pull <image> image：镜像名称
后台守护式运行镜像 docker run -d <image> image：镜像名称
运行镜像的同时进入交互界面 docker run -it <image> /bin/bash image：镜像名称
运行镜像的同时设置环境变量 docker -e <name>=<value> <image> name：环境变量名称，value：环境变量的值，image：镜像名称
运行的时候进行端口映射 docker run -p <hostport>:<containerport> hostport：主机端口，containerport：容器端口
运行容器的时候指定容器名称 docker run --name <name> name：所指定的容器的名称
运行容器的时候指定重启策略 docker run --restart=<val> val：restart策略的值，可选always和on-failure等值
进入正在运行的镜像 docker exec -it <containerid> /bin/bash containerid：容器id
停止运行容器 docker stop <containerid> containerid：容器id
删除容器 docker rm <containerid> containerid：容器id

注：以上表格中的尖括号<>所括的内容为必选参数，运行命令时必须替换成实际内容。
